class Perl6::Documentable::Search

Search entries generator.

class Perl6::Documentable::Search {}

This class takes a Perl6::Documentable::Registry and generate all the necessary entries to enable the search capabilities.

What is an entry? Is a JSON object with three different attributes:

{
    "category": "text displayed at the beginning",
    "value": "text displayed just under category",
    "url": "url to the document pointed by the entry"
}

method generate-entries

Defined as

method generate-entries(--> Array[Str])

Generates all search entries and store them in an Array. At this moment, it generates three kind of entries:

  • Primary entries: this entries comes from Perl6::Documentable::Primary objects. We initialize one entry per instance of that class found in the registry as follows:

    category => subkind attribute
    value    => name attribute
    url      => url attribute

  • Secondary entries: this entries comes from Perl6::Documentable::Secondary objects. Before create any entry of this type, we group all Perl6::Documentable::Secondary objects by name and after that, we generate one entry per group, as follows:

    category => kind attribute if more than one document is found. Subkind attribute otherwise
    value    => name attribute
    url      => url attribute

  • Reference entries: this entries comes from Perl6::Documentable::Index objects. One entry per found object.

    category => kind attribute (it's always Kind::Reference)
    value    => name attribute
    url      => url attribute
