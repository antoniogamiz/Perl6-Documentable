How to setup Perl6::TypeGraph

So you have an amazing hierarchy in the types defined in your module and wants to show it in your documentation? Perl6::TypeGraph will let you do it!

heaed2

TypeGraph file

At this moment, Perl6::TypeGraph is not able to automatically detect the hierarchy of a documented class, so you need to provide it with a file containing that hierarchy.

You two options to specify this file:

  • Use the --typegraph-file to specify a custom path.

  • Creates a file called type-graph.txt file in your module directory.

How to write this file? You need to follow a very easy syntax:

[name of the category]
# you can write comments too! 
Name::Of::A::Class is Parent::Class does A::ROle

You can check the type-graph.txt file used by Perl6::Documentable as an example.
